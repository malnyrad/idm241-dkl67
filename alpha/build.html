Triggers

  • Hover On
  • Hover On + On Icon
  • Hover On + Off Icon
  • Hover Off
  • On Click (Mouse Up)

Rules

Memo (Default)

  • Hover On
    • Cursor changes to finger.
    • Square area around icon quickly fades from white to light gray.
    • Rounded green bar quickly fades in on the right edge of the square area.
    • "Memo" tagline appears to the left of the square area.
  • Hover On + On Icon
    • Alt tag appears below the cursor.
  • Hover On + Off Icon
    • Alt tag below the cursor disappears.
  • Hover Off
    • Cursor changes back to pointer.
    • Square area around icon quickly fades back to white.
    • Rounded green bar quickly fades out.
    • "Memo" tagline disappears.
  • On Click (Mouse Up)
    • Side panel slides in from the right side of the screen.
    • Square area around icon changes from light gray to light green.
    • Rounded green bar remains active.

Memo (Selected)

  • Hover On
    • Cursor changes to finger.
    • "Memo" tagline appears to the left of the square area.
  • Hover On + On Icon
    • Alt tag appears below the cursor.
  • Hover On + Off Icon
    • Alt tag below the cursor disappears.
  • Hover Off
    • Cursor changes back to pointer.
    • "Memo" tagline disappears.
  • On Click (Mouse Up)
    • Side panel slides out to the right side of the screen.
    • Square area around icon changes from light green to white.
    • Rounded green bar disappears.
