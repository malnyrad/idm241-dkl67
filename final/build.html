Rules

Icon Sidebar (All)

  • Hover On
    • Cursor changes to finger
    • Background color changes to light grey
    • Rounded green bar appears on right edge
  • Hover On + 1s Delay
    • Tagline fades in on the left side of the sidebar
  • Hover Off
    • Cursor returns to default
    • Background color returns to white
    • Rounded green bar on right edge disappears
    • Tagline fades out on the left side of the sidebar
  • Mouse Down
    • Icon size decreases by 15%
    • Background color changes to dark grey
